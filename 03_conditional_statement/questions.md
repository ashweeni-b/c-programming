### Practice Questions

1. What will be the output of this program
    ``` c
    int a = 10;
    if(a = 11)
        printf("I am 11");
    else
        print("I am not 11");
    ```

    Solution: <br />
    The output of the program is: ```I am 11``` <br />
    The value of a becomes 11 which is a non-zero value and so the condition becomes true and the if statement gets executed.
