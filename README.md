## C Playground 🚀

A structured repository for practicing C programming - from fundamentals to problem-solving. 

This repository contains topic-wise C programs, and practice problems solved while building a strong foundation in programming.

Also it contains the problems solved on HackerRank and CodeChef platforms.

### 📂 Repository Structure

Each folder contains:
- 📄 C programs (.c files)
- 📝 README.md (Problem solved)

### 🧠 Topics Covered

1. Basics of Programming
1. Variables, Data Types, Input Output
1. Instructions and Operators
1. Conditional Instructions
1. Loop Control Instructions
1. Functions and Recursion
1. Pointers
1. Arrays 

### ⚙️ How to Run

In VSCode,

```bash
gcc filename.c 
./a.exe
```

### 🔥 Goals
- Build strong programming fundamentals
- Transition to Data Structures & Algorithms
- Develop problem-solving skills
